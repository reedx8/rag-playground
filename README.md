# RAG Playground

This is a [Convex](https://convex.dev/) project created with [`npm create convex`](https://www.npmjs.com/package/create-convex).

## Get started

```
git clone https://github.com/reedx8/rag-playground.git
cd rag-playground
npm install
npm run dev
```
